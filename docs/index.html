getStateCities

This function returns an array with the cities of the given state

getStateCities(stateObject: Object): stateType?
Parameters
stateObject (Object) The object the tells the name of the state.
Name Description
stateObject.state string (default requiredParam('state')) The state name.
Returns
stateType?
Example
const cities = api.getStateCities({ state: 'São Paulo' });
// { state: 'São Paulo', abbreviation: 'sp', cities: ['Santos', 'São Vicente', 'Guarujá',...] }
